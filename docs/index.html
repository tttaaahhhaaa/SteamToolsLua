What is SteamToolsLua?

SteamToolsLua is a desktop tool that automates downloading, extracting, and managing Online Fix game releases. It integrates with Steam, LuaTools, and CloudRedirect to provide a seamless experience for playing LAN / online multiplayer games without the official Steam servers.

  • Online Fix Download & extract game fixes from online-fix.me
  • SteamDB Browse Steam app database, view depots & manifests
  • Lua Injection Inject Lua scripts into Steam client to unlock features
  • CloudRedirect Fix "No internet connection" errors in games
  • AI Translation Translate game UI text using AI
  • Auto-Update One-click update to the latest version

Updates

The app checks for updates on every launch. If a new version is available, it will prompt you to download and install it automatically. No manual re-downloading needed.

Note: CloudRedirect is bundled inside the app. When you update, the latest CloudRedirect version is included automatically.

v1.9.0: CloudRedirectTR — Full-featured Turkish CloudRedirect (no update check). All original features preserved, Turkish UI.

CloudRedirect – No Internet Fix

Some Online Fix games require CloudRedirect to bypass the "No internet connection" / "Offline mode" error. This tool is embedded directly in SteamToolsLua. Simply use the CloudRedirect SS button inside the app to configure it per-game. No separate download needed.

Important The bundled CloudRedirect may not always be the latest version. If you encounter issues, check for an app update first.

Unlock Removed

The old "Unlock" function has been removed. All unlock-related functionality is now handled through CloudRedirect. If you previously used the Unlock feature, simply use the CloudRedirect option instead — it covers the same use cases and more.
